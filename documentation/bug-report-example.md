# Пример баг-репорта

## Заголовок
Приложение падает при вводе спецсимволов `#` в поле "Имя" при регистрации

## Метаданные
- **ID:** BUG-2026-001
- **Статус:** Новый
- **Приоритет:** Высокий (блокирующий)
- **Серьезность:** Критическая
- **Окружение:** Chrome 120, Windows 11, demoqa.com v2.3.1

## Шаги воспроизведения
1. Открыть страницу регистрации `/register`
2. Ввести в поле "Имя" значение `Иван#`
3. Заполнить остальные поля валидными данными
4. Нажать кнопку "Зарегистрироваться"
